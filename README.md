### DESCRIPCIÓN DE LA APP: 
<p>
Esta aplicación busca transmitir una explicación mediante el uso de fragments de los TextFields, botones, elementos de selección, listas y elementos de información, siendo estos elementos los mas básicos y comunes de encontrar, siendo acompañados mediante ejemplos interactivos para ver como se marcan y alguna cosa que puede hacer, asi como algunos ejemplos de donde se pueden encontrar para comprender mejor su funcionalidad y uso cotidiano.
</p>
<p>
Adentrándonos al código, se compone de 2 .XML principales, el activity que contiene el titulo, el spinner para poder cambiar de activity y el fragment, el cual se ocupa mediante otro .XML que contiene toda la información del elemento seleccionado, el cual varia dependiendo el elemento, pero siempre ocupa un scrollview por si la información es demasiada pueda verse siempre de forma correcta. Sobre su parte lógica, también tiene 2, una que es para el activity el cual contiene código para su funcionamiento, un código que llama el fragment y la lógica para poder ocupar el spinner de forma adecuada, y el otro el fragment, el cual contiene código para que funcione el .XML, pero también tiene la lógica de lo que hará el contenido interactivo, por ejemplo, en los botones tenemos si mostrara un toast que es lo que dice, o por ejemplo en el switch el código para cambiar de color.
</p>

---

### INSTRUCCIONES DE USO: 
1. Abrir Android Studio.
2. Seleccionar clone repository
3. Ingresar la URL del repositorio en donde solicita el link
4. Dar click en clone
5. Esperar a que clone el repositorio
6. Seleccionar un dispositivo para ejecutar la aplicación
7. Dar en Run para correr la aplicación
8. Una vez que abra la aplicación, dar click en seleccionar una actividad
9. Hacer lo mismo hasta ver todas las activity disponible
